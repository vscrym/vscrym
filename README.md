Olá, viajantes! Me chamo João Vitor Santos, mais conhecido como "Scrym", e sou do Brasil. Estou iniciando minha jornada como desenvolvedor e sou apaixonado por tecnologia. No momento, estou aprendendo os fundamentos da programação e ampliando meus conhecimentos passo a passo. Meu objetivo é me tornar um Desenvolvedor Full Stack, trabalhando com React no front-end e Java + Spring Boot no back-end. Estou sempre em busca de novos desafios, evoluindo constantemente e transformando cada aprendizado em oportunidade.
E, acima de tudo, sigo firme com a mesma visão que me impulsiona desde o início: criar soluções que façam diferença e construir, com cada linha de código, o futuro que sonho alcançar.
